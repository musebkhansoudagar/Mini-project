# Reverse Engineering of Kick Cycle

A mini project focused on the systematic reverse engineering, CAD reconstruction, and finite element analysis of a commercially available kick cycle. This project aims to understand the design, material selection, structural behavior, and safety margins of lightweight personal mobility systems.

## ğŸ›  Methodology

1. **Product Selection & Inspection**  
   - Selection of a commercially available kick cycle  
   - Preliminary functional assessment  

2. **Disassembly & Measurement**  
   - Complete disassembly of components  
   - Dimensional measurement using vernier calipers, micrometers, and steel rulers  

3. **CAD Modeling & Drafting**  
   - 3D modeling of individual components using **CATIA V5R20**  
   - Preparation of 2D engineering drawings  

4. **Finite Element Analysis (FEM)**  
   - Static structural analysis under operational load  
   - Explicit dynamic analysis for high-impact conditions  

5. **Evaluation & Validation**  
   - Interpretation of stress, strain, and deformation results  
   - Safety margin assessment
